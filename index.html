OWNER & FOUNDER · P.A.L. PREMIUM ACTIVITIES & LIFESTYLE

With over 7 years of experience managing complex operations across technology and e-commerce companies, I built a career around one core skill: making systems run smoothly, efficiently, and without friction.

I have led logistics and operations teams, managed multiple business units simultaneously, coordinated cross-functional projects, and developed processes from the ground up at companies including Uber, Insider Store, and Merama, working across Brazil and internationally.

The years between 2024 and 2025 were a period of recalibration. New challenges, limited working hours, and the language barrier guided me toward frontline roles, which turned out to be incredibly valuable. I gained hands-on experience across a wide range of everyday tasks, building a practical understanding of what it truly takes to keep a home, a schedule, and a life running smoothly.

I know what it feels like to be a busy professional myself. For years, I worked long hours doing what I loved, often with little time left to care for the things that mattered most to me personally. That experience shaped the way I see this work today. I believe I can now help give that time back to others, the same time I once wished I had more of.

P.A.L. was born in early 2026, built on everything I had studied, led, and lived. I also co-host Airbnb properties in Sydney, consistently achieving top guest ratings through the same attention to detail and hands-on approach I bring to every client engagement.

const SHELL_ASSETS = [ '/', '/index.html', '/pal-logo-black.jpeg', '/luz_portrait_bw_pal_badge.png' ]; // ── INSTALL: cache shell assets ─────────────────────────────────── self.addEventListener('install', event => { event.waitUntil( caches.open(CACHE_NAME).then(cache => { return cache.addAll(SHELL_ASSETS); }).then(() => self.skipWaiting()) ); }); // ── ACTIVATE: delete old caches ─────────────────────────────────── self.addEventListener('activate', event => { event.waitUntil( caches.keys().then(keys => Promise.all( keys .filter(key => key !== CACHE_NAME) .map(key => caches.delete(key)) ) ).then(() => self.clients.claim()) ); }); // ── FETCH: network-first for API, cache-first for shell ────────── self.addEventListener('fetch', event => { const url = event.request.url; // Always go network-first for Google APIs and Apps Script if ( url.includes('script.google.com') || url.includes('docs.google.com') || url.includes('accounts.google.com') || url.includes('onesignal.com') ) { event.respondWith( fetch(event.request).catch(() => caches.match(event.request)) ); return; } // Cache-first for shell assets (html, images, etc.) event.respondWith( caches.match(event.request).then(cached => { if (cached) return cached; return fetch(event.request).then(response => { // Only cache valid same-origin GET responses if ( !response || response.status !== 200 || response.type === 'opaque' || event.request.method !== 'GET' ) { return response; } const toCache = response.clone(); caches.open(CACHE_NAME).then(cache => cache.put(event.request, toCache)); return response; }).catch(() => { // Offline fallback: return cached index.html if (event.request.mode === 'navigate') { return caches.match('/index.html'); } }); }) ); }); // ── PUSH NOTIFICATIONS ──────────────────────────────────────────── self.addEventListener('push', event => { const data = event.data ? event.data.json() : {}; const title = data.title || 'P.A.L. Premium Activities & Lifestyle'; const body = data.body || 'You have a new notification.'; const icon = data.icon || '/pal-logo-black.jpeg'; const badge = data.badge || '/pal-logo-black.jpeg'; const url = data.url || '/'; event.waitUntil( self.registration.showNotification(title, { body, icon, badge, data: { url }, vibrate: [200, 100, 200] }) ); }); // ── NOTIFICATION CLICK ──────────────────────────────────────────── self.addEventListener('notificationclick', event => { event.notification.close(); const target = event.notification.data?.url || '/'; event.waitUntil( clients.matchAll({ type: 'window', includeUncontrolled: true }).then(list => { for (const client of list) { if (client.url === target && 'focus' in client) return client.focus(); } if (clients.openWindow) return clients.openWindow(target); }) ); });
